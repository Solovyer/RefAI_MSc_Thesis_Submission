# output

This folder contains an example run folder from the current evaluated pipeline version.

The most important subfolders are:

- `Reviewed_Outputs_01`: main readable extraction outputs.
- `Llm_Review_Outputs_02`: technical LLM review/audit outputs.
- `Quality_Control_00`: batch summaries and suspect records.
- `Raw_Outputs_03` and `Logs_04`: detailed run outputs and logs for auditability.
