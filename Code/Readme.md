# code

This folder contains the runnable RefAI pipeline code and the matching notebook copy.

- `Refai_V7_1_1_Slm_Local_Dating.py` is the recommended runnable script for the deposition package.
- `Refai_V7.ipynb` is the notebook version used during development.

The script is prepared for deposition: `PROJECT_DIR` defaults to the package root, and can be overridden with the `REFAI_PROJECT_DIR` environment variable. API keys are not stored in the code.
